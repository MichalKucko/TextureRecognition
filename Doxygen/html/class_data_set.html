Detailed Description

Class used for preparing dataset containing samples of features and corresponding class labels. By default it is assumed, that images, from which the set is created, are placed in the location of executable or "main.c" file in folder called "database" and then in folders associated with different classes. For example, if in "database" folder there are two folders called "A" and "B", images in these folders represent two classes - A and B.
